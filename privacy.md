# ギルドノート プライバシーポリシー

最終更新日: 2026年3月19日

---

ギルドノート（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリにおける情報の収集、使用、保護について説明するものです。

---

## 1. 収集する情報

本アプリでは、以下の情報を収集する場合があります。

### アカウント情報（保護者）

- メールアドレス（メール/パスワード認証の場合）
- 認証プロバイダ情報（Google、Apple でのログインの場合）

### アカウント情報（お子さま）

- お子さまのデバイスは招待コードによる匿名認証で参加します。メールアドレスやパスワードは収集しません。

### アプリ内データ

- ギルド名、メンバー名、年齢
- クエスト（タスク）の内容・達成履歴
- レベル、経験値などのゲーム進捗データ
- ギルドラン（ミニゲーム）の挑戦履歴・スコア・獲得経験値

### 自動収集データ

- アプリの利用状況（Firebase Analytics）
- クラッシュログ（Firebase Crashlytics）
- デバイス情報（OS、デバイスモデル）

## 2. 情報の利用目的

収集した情報は、以下の目的で利用します。

- 本アプリのサービス提供および機能改善
- ユーザーサポートへの対応
- アプリの安定性向上およびバグ修正
- 利用状況の分析によるサービス改善
- 通知の送信（ユーザーが許可した場合）
- 進捗レポート（PDF）やギルドカード（画像）のエクスポート機能の提供

## 3. 第三者への情報提供

運営者は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

- ユーザーの同意がある場合
- 法令に基づく開示要求がある場合
- サービス提供に必要な業務委託先への提供（適切な管理のもと）

## 4. Firebase・外部サービスの利用

本アプリは、Google が提供する Firebase サービスを利用しています。

- Firebase Authentication（認証 — メール/パスワード、Google、Apple、匿名認証）
- Cloud Firestore（データ保存）
- Firebase Analytics（利用分析）
- Firebase Crashlytics（クラッシュ分析）
- Firebase Cloud Messaging（プッシュ通知）

これらのサービスにおけるデータの取り扱いについては、[Google のプライバシーポリシー](https://policies.google.com/privacy) をご参照ください。

## 5. データの保護

運営者は、収集した情報の漏洩、紛失、改ざんを防止するため、合理的な技術的・組織的セキュリティ対策を講じます。

- 通信の暗号化（SSL/TLS）
- Firebase のセキュリティルールによるアクセス制御
- パスコード機能によるモード切替・メンバー切替の保護
- データはデバイス上（ローカルストレージ）とクラウド（Firestore）に保存されます

## 6. お子さまのプライバシー

本アプリは、保護者の管理のもとでお子さまが使用することを前提としています。

- 13歳未満のお子さまの個人情報は、保護者の同意のもとで収集されます。
- お子さまのデバイスは招待コードによる匿名認証で参加するため、お子さまから直接メールアドレスやパスワードを収集することはありません。
- お子さまの利用データ（クエスト履歴、ギルドラン履歴、レベル等）は保護者のアカウントに紐づいて管理されます。
- お子さまのデバイスでは Firebase Analytics によるデータ収集を無効化しています。
- アプリ内課金を行う際は、保護者確認のためのペアレンタルゲート（認証画面）を設けています。
- 保護者は、お子さまのデータの閲覧・削除をアプリの設定画面から行うことができます。
- 保護者がアカウントを削除した場合、お子さまのデータも含めすべてのデータが削除されます。

## 7. データの削除

ユーザーは、以下の方法でデータを削除できます。

- アプリ内の設定画面「アカウント削除」から、アカウントおよび全データを削除
- お子さまのデバイスから「ギルドから退出」でローカルデータを削除
- 個別のデータ削除をご希望の場合は、下記連絡先までお問い合わせください

アカウント削除後、関連するすべてのデータ（ギルドデータ、クエスト履歴、ギルドラン履歴、メンバー情報等）は完全に削除され、復元することはできません。

## 8. プライバシーポリシーの改定

運営者は、必要に応じて本プライバシーポリシーを改定することがあります。重要な変更がある場合は、アプリ内での通知またはその他の適切な方法でお知らせします。改定後のプライバシーポリシーは、本アプリ内に掲示した時点から効力を生じます。
